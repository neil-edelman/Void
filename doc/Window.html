Entry-point. Deals with providing an interface to the libraries.

Uses the OpenGL API for graphics, https://www.opengl.org/. On some machines, you have to query the library for OpenGL2+. If GLEW is defined by the environment, it uses this, http://glew.sourceforge.net/. (Necessary for Windows, Linux.)

FreeGLUT, http://freeglut.sourceforge.net/, is used for the cross-platform window-manager. However, this is built-in 1998-GLUT-compatible for Mac: FreeGLUT uses XQuartz or X11, which is absolutely unacceptable.
