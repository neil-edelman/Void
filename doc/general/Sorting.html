bubble

int bubble (void **base_ptr, int (*compare)(const void *, const void *), void **(*address_next)(void *const))

Iteration-separated linked-list bubble sort; for when we don't have to get the right answer right away. Works with a singly-linked list.

parameter: base_ptr
Pointer to the first element or null.
parameter: compare
Compares two elements; switches them if it returns positive.
parameter: address_next
Given an item, returns the address of the next item. Must be valid.
