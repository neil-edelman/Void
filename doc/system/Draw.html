enum TexClass {
	TEX_CLASS_SPRITE,
	TEX_CLASS_NORMAL,
	TEX_CLASS_BACKGROUND
}

Texture addresses on the graphics card, when you want different textures simultaneously. Limit of MAX_COMBINED_TEXTURE_IMAGE_UNITS, which may be small, (like 2?)

enum

enum { /* vec2 */ VBO_ATTRIB_CENTRED, /* vec2 */ VBO_ATTRIB_TEXTURE }

Shader attribute assignment. VBO_ATTRIB_CENTERED is the normalised centre co-ordinates in which the sprite goes from [-1, 1]; VBO_ATTRIB_TEXTURE maps that to [0, 1] for texture mapping the sprite.
